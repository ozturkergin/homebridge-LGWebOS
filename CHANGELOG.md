# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-07-22
### Changed
- Improved input source visibility and static naming.

## [1.0.0] - 2026-07-22
### Added
- Initial release as a fork of `homebridge-webos-tv`.
- Added support for statically defining inputs in configuration (`config.inputs`), giving feature parity with `homebridge-yamaha-avr` input handling.
